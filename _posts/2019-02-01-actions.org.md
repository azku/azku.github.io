## Camping Exchange

With elixir and phoenix and npm locally installed do:

{% highlight shell %}
mix phx.new camping_exchange
{% endhighlight %}

-   Click "Y" when asked: "Fetch and install dependencies?"

-   Ones the installation is successfull
    
    {% highlight shell %}
    cd camping_exchange
    touch docker-compose.yml
    {% endhighlight %}
